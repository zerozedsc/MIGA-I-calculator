## Overview

Single-page app built with Vite, React, and TypeScript.

## Important files

- `index.html` — HTML shell; must contain the DOM mount element.
- `index.tsx` — React bootstrap (creates root and mounts the app).
- `App.tsx` — main UI composition.
- `vite.config.ts` — Vite configuration.

## Folders

- `components/` — UI components
- `utils/` — calculation + translation utilities
